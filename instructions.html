The Techniques

The Keyboard Technique is the use of the shortcuts Cntrl-C and Cntrl-V to copy and paste text, respectively.

The AutoComPaste Technique is the use of autocomplete software to copy-paste text.

AutoComPaste

To use AutoComPaste

  1. Type three characters from the text you wish to copy
  2. A list of matching text will appear
  3. As you type more characters from the text in question, the matching text list will refine accordingly
  4. Select a text component to copy-paste using the arrow keys and the enter button
  5. Once a text component is selected, the right arrow key can be used to copy-paste the next text component
  6. The left arrow key can be used to remove any unwanted text components added after the first
  7. Once you are satisfied, click the Next Task button on the left hand side to continue to the next trial
